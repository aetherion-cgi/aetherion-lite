## 💡 Key Concepts

### Parent Agent
Orchestrates the entire analysis process
- Loads manifests
- Spawns child agents
- Coordinates integrations
- Validates with governance

### Child Agents
Specialized micro-reasoners (ephemeral)
- Market sizing
- Competitive analysis
- Regulatory risk
- Financial forecasting
- Technology assessment
- Customer analysis
- Team evaluation

### Manifests
YAML configuration files (no code)
- Define metrics per industry
- Specify risk factors
- Configure child agents
- Set governance thresholds

### Integration Layer
Connects to Aetherion platform
- UDOA: Data orchestration
- URPE: Strategic analysis
- UIE: Intelligence synthesis
- Governance: Constitutional validation
- ILE: Meta-learning
