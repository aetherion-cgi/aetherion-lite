## 🚀 QUICK START (3 COMMANDS)

1. **Copy files to your repo:**
```bash
cd /path/to/Aetherion-Master
cp -r /path/to/aetherion-uie-integration/* universal-intelligence-engine/
```

2. **Setup & deploy:**
```bash
# Create .env
cat > .env << 'EOF'
AETHERION_API_KEYS=demo-key-12345,customgpt-key,claude-key
POSTGRES_PASSWORD=change-me
NEO4J_PASSWORD=change-me
EOF

# Generate certs & deploy
./scripts/generate-certs.sh  # (script in MIGRATION_GUIDE.md)
docker-compose up -d
```

3. **Test:**
```bash
curl http://localhost:8000/health
python aetherion_smoke_test.py
```

   - Name: "Aetherion"
   - Auth: API Key (X-API-Key header)
   - Base URL: `http://your-domain:8000`
   - Schema: `/openapi.json`
